# Print Fornece ERP

Sistema de gestão para DTF Têxtil, DTF UV e Camisa Lisa.

## Módulos
- ✅ Frente de Caixa B2B (com tabela de preços por volume)
- ✅ Financeiro (contas a pagar e receber)
- 🔜 Estoque
- 🔜 Pedidos
- 🔜 Clientes

## Deploy na Vercel
1. Faça fork ou importe este repositório na [Vercel](https://vercel.com)
2. Clique em "Deploy"
3. Pronto — sistema disponível em qualquer navegador

## Desenvolvimento local
```bash
npm install
npm run dev
```
